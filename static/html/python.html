递归函数

xxxxxxxxxx
1、在函数内部调用自己
2、需要设置递归的边界（出口，退出循环）

示例: 阶乘

xxxxxxxxxx
def rec(n):
    if n == 1:
        return 1 # 做为递归的出口,不然死循环(注意: 这个1 要和4运算: 4*1, 不能乱return )
    return  n * rec(n-1) # 4 * (3,2,1) = 12+8+4 = 24
​
res=rec(4)
print(f"结果为: {res}"
​

示例: 斐波那契数列

规律: 从第三个数开始,前面两个相加的和等于第三个数: 11235813

xxxxxxxxxx
      (n-1) + (n-2) = n (前一个+ 前前个 = 当前数)
xxxxxxxxxx
# 规律: n-1 + (n-2)
def fibo(n):
    if n==1 or n==2:
        return 1
    return fibo(n-1) + fibo(n-2)
​
res=[]
n=int(input("斐波那契数列: "))
for i in range(1,n+1):
    res.append(fibo(i))
​
print(res)

闭包

概述

满足两个条件,就叫闭包

  1. 函数内部再定义一个函数, 并且return出去
  2. 内部第二个函数对外部函数变量的引用(非全局变量)

 

作用: 延长局部变量的生命周期

 

闭包的应用

xxxxxxxxxx
# 计算一元一次方程
def line_config(a,b):
    def count(x):
        return a*x+b
    return count
​
c = line_config(1,1) # 这个1,1将被保存,延长生命周期
print(c(0))
print(c(1))
​

要理解什么是多态，我们首先要对数据类型再作一点说明

def run_twice(animal):

    Animal is running...
​
>>> run_twice(Dog())     # 实例化Dog,传给函数
    Dog is running...    
    
>>> run_twice(Cat())     # 实例化Cat,传给函数
    Cat is running...    
    
# PS 如果传入的类实例有run方法就执行,没有就执行父类Animal的run方法
对于静态语言（例如Java）来说，如果需要传入Animal类型，则传入的对象必须是Animal或子类
对于Python这样的动态语言来说，我们只需要保证传入的对象有一个run()方法就可以了
这就是动态语言的“鸭子类型”，它并不要求严格的继承体系，一个对象只要“看起来像鸭子，走起路来像鸭子”，那它就可以被看做是鸭子。

高级特性

高级特性-生成器

概述

  1. 列表太大很耗内存, 生成器保存一种算法推导列表下一个元素.

  2. 计算到最后一个元素，没有更多的元素时,抛出StopIteration的错误

  3. generator 是可以迭代的对象,我们使用时一般都用for迭代, 不需要关心StopIteration的错误。

  4. 生成器有两个方法: next() 和 x.send(None)

  5. 关键字: yield, 代码执行到这里的时候会暂停, next()或send后,继续执行, 然后再次暂停

    yield 相当函数的 return , 返回一个变量, 然后暂停, 下次调用next或send继续执行后面代码

  6. 对于函数式generator遇到return 或执行到函数体的最后一行,结束generator

    注意: 普通函数调用,直接返回结果, generator调用,返回一个对象

 

使用列表生成式

xxxxxxxxxx
# 将列表的[] 换成 () 即可变成生成器

​
# 使用for 迭代(注意,generator没有len属性)
for n in g:
    print(n)

def fib(max):
    n,a,b = 0,0,1

        yield b
        a,b=b,a+b
        n = n+1
    return "done"
